Il n'est possible de tracer que des rectangles.
On ne peut sélectionner que le dernier rectangle.
Pour déplacer le dernier rectangle crée il faut cliquer dessus et dragger la souris.
Pour changer la couleur du dernier rectangle crée il faut choisir une nouvelle couleur dans le ColorPicker puis cliquer sur le rectangle en question.
Le bouton delete supprime le dernier rectangle crée.

version du jdk utilisé : 11.0.2
